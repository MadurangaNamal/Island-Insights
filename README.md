# Island Insights

Island Insights is a Blazor WebAssembly application built with .NET 9. It provides interactive news and information about islands, leveraging modern web technologies for a fast and responsive user experience.

## Features

- Latest island news and updates
- Interactive components powered by Blazor
- Fast client-side performance with WebAssembly
- Modular architecture for easy extension

## Getting Started

### Prerequisites

- [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0)
- Modern web browser

### Running the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/MadurangaNamal/Island-Insights.git
   ```

2. Navigate to the project directory:

   ```bash
   cd "Island Insights"
   ```

3. Run the application:

   ```bash
   dotnet run
   ```

4. Open your browser and go to `http://localhost:5190` (or the port shown in the console).
